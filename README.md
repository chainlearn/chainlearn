# ChainLearn

ChainLearn is an open-source, blockchain-based learning platform that allows users to take courses, securely pay mentors via escrow, and earn NFT certificates upon completion. It combines modern web technologies with smart contracts on the Stellar network.

## Features

- **Escrow Payments:** Studentsâ€™ payments are held securely until course completion.
- **NFT Certificates:** Learners receive verifiable proof of course completion as NFTs.
- **Full-Stack Web App:** Includes frontend, backend, and smart contracts.
- **Open Source:** Fully transparent and extendable.

## Tech Stack

- **Frontend:** Next.js, Tailwind CSS, TypeScript  
- **Backend:** NestJS, PostgreSQL  
- **Smart Contracts:** Rust, Soroban (Stellar smart contracts)
